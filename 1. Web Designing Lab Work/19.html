JavaScript Predefined Functions

Array Functions

The following are some of the most commonly used array functions in JavaScript:

  • push(): Appends a value to the end of an array.
  • pop(): Removes the last element from an array.
  • shift(): Removes the first element from an array.
  • unshift(): Inserts a value at the beginning of an array.
  • indexOf(): Returns the index of the first occurrence of a value in an array.
  • lastIndexOf(): Returns the index of the last occurrence of a value in an array.
  • concat(): Concatenates two or more arrays.
  • slice(): Returns a slice of an array.
  • join(): Joins the elements of an array with a separator.

Date Functions

The following are some of the most commonly used Date functions in JavaScript:

  • constructor(): Creates a new Date object.
  • getDate(): Returns the day of the month (from 1-31).
  • getMonth(): Returns the month (from 0-11).
  • getFullYear(): Returns the year.
  • getHours(): Returns the hour (from 0-23).
  • getMinutes(): Returns the minutes (from 0-59).
  • getSeconds(): Returns the seconds (from 0-59).
  • getMilliseconds(): Returns the milliseconds (from 0-999).
  • getTime(): Returns the number of milliseconds since midnight Jan 1 1970.
  • toLocaleString(): Returns a localized string representation of the date.
  • toUTCString(): Returns a UTC string representation of the date.
